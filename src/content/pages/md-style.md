表格样式:

| 字段          | 描述                                                  |
| :------------ | :--------------------------------------------------- |
| 姓名 (NAME)   | 显示在页眉和页脚。用于SEO和RSS。                     |
| 邮箱 (EMAIL)  | 显示在联系信息部分。                                  |
| 文章数量 (NUM_POSTS)    | 限制首页显示的文章数量。                              |
| 作品数量 (NUM_WORKS)    | 限制首页显示的作品数量。                              |
| 项目数量 (NUM_PROJECTS) | 限制首页显示的项目数量。                              |

代码片段:

```ts
export const data = {
  name: '名称',
  value: '值'
}

export function getName() {
  return data.name
}
```
